# HTMl

 - hypertext markup language
 - used to design the web UI

Markup Language
    - which is made up og 
        - tag 
            word enclosed by < and >
        - also know as element 
        - e.g 
            - <p>
            - <table>
        -  used to perform some operation
    -types
       - opening 
            - at start
            e.g: <p>, <div>
       - closing 
            - at the end 
             e.g: </p>, </div>
        - empty 
            - without having any data
            e.g: <br/>
        - root
            - which starts abd ends the document
            e.g: <html>
